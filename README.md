### 기술

- XML 기반의 기본적인 안드로이드 네이티브 앱 개발

- SQLite, SharedPreferences 사용
- ViewPager, RecyclerView 사용
- 구글 광고 삽입
- 플레이스토어에 배포 후 유지 보수 중
